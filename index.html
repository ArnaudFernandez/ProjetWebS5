Commandes :
- Flèches directionnelles : Déplacer le curseur (Carré jaune)
- Retour chariot : Sur base -> Ouvre le menu de la base // Sur unité -> Selectionne l'unité // Sur unité ennemie quand unité selectionné -> Attaque
- Echap : Menu de base ouvert -> Ferme le menu // Unité sélectionné -> Désélectionne l'unité // Aucun des deux -> Passe son tour

Les joueurs se voient créditer de 15 d'argent à chaque début de tour.
Un déplacement par tour par unité
Une attaque par tour par unité
